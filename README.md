# -2026

Kotlin/JS веб-додаток — шаблон проєкту.

Вимоги:
- JDK 11+ (рекомендується 17)
- Gradle або Gradle Wrapper

Команди:
- `gradle browserDevelopmentRun` — запустити dev сервер і відкрити додаток у браузері
- `gradle build` — зібрати production-бандл

Структура додатку:
- `build.gradle.kts` — конфігурація Gradle/Kotlin/JS
- `src/main/kotlin/Main.kt` — основний код
- `src/main/resources/index.html` — HTML-шаблон
